How to Run:
Clone the repository and open the solution in Visual Studio
Restore packages via Tools > Nuget Package Manager > Manage Nuget Packages for Solution
Build the solution via Build > Build 9gbScale
Run the solution via Debug > Start Debugging
Enter anything into the console window to start the application, it will print which bar is fake
The application will pause after it has selected the fake bar, enter into the console window again to continue
Press 1 to refresh the page and do another test, or any other key to exit the program
